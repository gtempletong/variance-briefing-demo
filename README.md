# Demo — Control de gestión (Constructora Vista Andes)

Dashboard ejecutivo + informe narrativo de **desvío presupuestario** (presupuesto vs real),
generado con IA a partir de una planilla. **Datos de ejemplo ficticios** — demo de
consultoría de Gavin Templeton.

- `index.html` — dashboard ejecutivo (KPIs, desvíos, bridge volumen/precio, timing vs estructural)
- `informe.html` — el informe narrativo del desvío

Servido como sitio estático en GitHub Pages.
